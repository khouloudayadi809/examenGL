Chapitre 2 : Routage des paths


Introduction
Dans un framework front-end (comme Angular), il n'est pas souhaitable de perdre tout son contexte à chaque changement de page.

Angular a donc besoin d'un systeme de routage pour permettre la navigation à travers différentes pages de votre application.

Pour realiser ce routage, Angular propose le module RouterModule disponible dans la librairie @angular/router

 
import{ NgModule } from '@angular/core';
import { Routes, RouterModule } from '@angular/router';
const routes: Routes = [];
@ngModule({
    imports :[RouterModule.forRoot(routes)],
    exports :[RouterModule]
})
export class appRout ingModule { }
